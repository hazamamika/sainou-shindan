# 好きなことで稼ぐ「才能」診断

生年月日と6つの質問に答えるだけで、あなたの才能・合った稼ぎ方・「これまでうまくいかなかった本当の理由」を読み解く診断アプリです。

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバーもデータベースも不要で、入力された内容はどこにも送信されません。

## 診断の仕組み

- **9つの気質**（生年月日の数字から算出）— 才能・合った働き方・人生の傾向
- **5つのエネルギータイプ**（生まれ年から算出・立春調整あり）— スランプ傾向とうまくいかない原因
- **6つの質問**（仕事・収入・人間関係・自由度・自分らしさ・挑戦の現在地）— 現状と理想のギャップを「理想の実現度%」として可視化

## カスタマイズ

- LINE誘導ボタンのリンク先：`index.html` 内の `const LINE_URL` を書き換え
- 診断文の変更：`TYPES`（9気質）と `ELEMENTS`（5タイプ）の各テキストを編集

---

※この診断の結果は人生を保証するものではありません。
